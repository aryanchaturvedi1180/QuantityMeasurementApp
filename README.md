# QuantityMeasurementApp

## Overview

This repository documents the daily progress of tasks completed during the BridgeLabz Training Program.

The project focuses on building a **Quantity Measurement Application** using Java by following clean coding principles and Test-Driven Development (TDD).

Progress is recorded **week-wise and day-wise**, covering multiple Use Cases (UCs) implemented using separate feature branches.

## 🎯 Use Case Details

### UC1 – Feet Equality

* Compare two measurements in feet
* Validate equality logic
* Implement unit test cases

### UC2 – Feet & Inch Equality

* Support comparison between feet and inches
* Convert units before comparison
* Ensure proper object-oriented implementation

### UC3 – Generic Quantity Class (DRY Principle)

* Create a generic `Quantity` class
* Eliminate duplicate logic
* Apply DRY (Don't Repeat Yourself) principle

### UC4 – Extended Unit Support

* Add support for additional units like Yard
* Improve scalability
* Enhance conversion flexibility

### UC5 – Unit Conversion

* Implement full unit-to-unit conversion
* Enable flexible and reusable measurement operations
